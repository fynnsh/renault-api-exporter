# Python API to query Renault Zoe information.

# Installation
```
virtualenv -v venv -p python3
pip install -r requirement.txt
```

# Launch
```
export RENAULT_USER=<myRenaultUser>
export RENAULT_PASS_=<myRenaultPass>
export RENAULT_VIN=<VIN>
python main.py
```
